# Machine Learning

Esta área contém estudos relacionados ao Aprendizado de Máquina, um campo de estudo que une matemática, estatística e programação para permitir que máquinas aprendam com dados e/ou realizem tarefas de forma autônoma.

Inicialmente, utilizarei bibliotecas prontas para realizar a implementação dos algoritmos de ML. Mais à frente, quero implementar os algoritmos manualmente.
